Execution

First make sure the is a SQL table named ex4 running on a separate server.
Then run from IntelliJ this 'Ex4Application, and go to localhost:8080


Undefined Characteristics:

The Messages are held in SQL database. Access to them is by using beans (singleton), in the appropriate places.

The list of active users is held in the Application Scope also by using a singleton bean.
Active users are maintained by "timestamping" this data structure everytime a user communicates with the server, and every 9 seconds this data structure removes users who didn't communicate with the server for over 10 seconds.

Bonus

A user cannot sign in with a name thats already taken.

Asynchronous Behavior

Active Users singleton's access methods are synchronized due to concurrent use by potential users.
